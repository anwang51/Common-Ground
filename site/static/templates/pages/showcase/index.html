Fall 2017 Project Descriptions

MuseCage

Project Leader: Nipun Ramakrishnan

Our team has implemented a visual question answering and music information retrieval system using a combination of convolutional neural networks, recurrent neural networks, and signal processing techniques.

DeepBeat

Project Leader: Arsh Zahed

We've implemented neural nets that write original music. This team has extended off of work done by Google Brain, incorporating a new neural net architecture known as Phase Functioned Neural Networks to emphasize structure and expression.

UAV Robotics

Project Leader: Hank O'Brien

The robotics team has converted an algorithm for ground based navigation and collision avoidance to work with quadcopters, training them in simulation to fly indoors while avoiding objects based on camera input and reinforcement learning.
